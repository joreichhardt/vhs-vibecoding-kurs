# Fun Facts zu KI und Coding

## Programmieren war immer Werkzeugbau

Viele Programme entstehen, weil Menschen wiederkehrende Aufgaben vereinfachen wollen. KI macht diesen Einstieg fuer kleine Werkzeuge sichtbarer und schneller.

## Webseiten sind gute Lernobjekte

HTML-Dateien lassen sich oft direkt im Browser oeffnen. Dadurch sieht man schnell, was der Code bewirkt.

## Fehlermeldungen sind Daten

Eine Fehlermeldung ist keine Niederlage, sondern ein Hinweis. Fuer Codex ist eine genaue Fehlermeldung oft der beste Ausgangspunkt fuer Hilfe.

## KI kann erklaeren und umbauen

Ein nuetzlicher Einsatz ist nicht nur "Code schreiben", sondern auch:

- Code in einfacher Sprache erklaeren
- eine komplizierte Loesung vereinfachen
- Testfaelle vorschlagen
- Texte fuer Benutzer verbessern
- eine Projektidee strukturieren

## Kursnutzen

Fun Facts eignen sich gut, um Hemmungen abzubauen: Coding ist nicht nur Mathematik und Spezialwissen, sondern auch Sprache, Struktur, Ausprobieren und Verbesserung.
