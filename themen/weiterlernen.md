# Weiterlernen nach dem Kurs

## Ziel

Die Teilnehmer kennen sinnvolle naechste Schritte, um nach dem Kurs eigenstaendig weiterzuarbeiten.

## Naechste technische Themen

- HTML und CSS Grundlagen
- JavaScript fuer Interaktion
- Dateien und Ordner im Projekt
- Browser-Entwicklertools
- Git als Versionskontrolle
- Markdown fuer Dokumentation

## Arbeitsgewohnheiten

- Projektideen klein starten
- Aenderungen sichern
- Prompts und Ergebnisse dokumentieren
- Fehlermeldungen sammeln
- regelmaessig aufraeumen
- Datenschutz vor dem Teilen pruefen

## Lernpfade

Moegliche Vertiefungen:

- Webseiten bauen
- Automationen mit Python
- Daten aus Tabellen verarbeiten
- einfache Apps mit Webtechnologien
- digitale Selbstorganisation

## Praktischer Rat

Nach dem Kurs sollte jedes neue Projekt mit einem kurzen Steckbrief beginnen. Danach wird zuerst eine Minimalversion gebaut, getestet und erst dann erweitert.
